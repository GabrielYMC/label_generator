\# 淡江大學課外活動組 - 器材標籤產生器

\*\*(TKU SA - Equipment Label Generator)\*\*



本專案為一款純前端 (Frontend-only) 的網頁應用程式，專為淡江大學課外活動組 (Tamkang University Student Activities Section) 設計，用於快速、自動化地產生具備 NERV/EVA 工業機甲風格的器材管理標籤。



系統支援單筆建立與 CSV 批次匯入，並內建「待列印清單 (Print Queue)」功能，允許使用者混搭不同版型與尺寸，最終輸出至 A4 紙張進行高保真 PDF 列印。



\---



\## 🚀 核心功能模組 (Core Features)



\### 1. 雙主視覺版型與自動縮放

\* \*\*Type A (橫式排版)\*\*：預設 15cm (大) 與 10cm (小)。採用雙列三欄緊湊排版，適合貼於長條型器材（如：麥克風、線材箱）。

\* \*\*Type B (方型排版)\*\*：預設 15cm (大) 與 10cm (小)。採用三列式堆疊排版，具備強烈視覺張力，適合大型設備或外箱（如：擴音機、投影機）。



\### 2. 參數化排版引擎 (Parametric Design)

提供使用者像素級 (Pixel-perfect) 的版面控制權，且所有變動皆為 \*\*即時渲染 (Live Preview)\*\*：

\* \*\*全域自訂\*\*：支援自訂單位 Logo 上傳、主副標題修改、底部警告標語自訂。

\* \*\*微調拉桿\*\*：可即時調整外框粗細、大/中/小字體級別、各方塊的水平與垂直間距。

\* \*\*獨立佔比分配\*\*：透過拉桿動態分配第一行與第二行的 Flexbox 寬度百分比 (%) 與高度佔比，確保任何長度的器材名稱與編號都不會跑版。



\### 3. 待列印清單與狀態快照 (Print Queue \& Snapshot)

導入類似「購物車」的概念。

\* 當使用者點擊「加入清單」時，系統會拍下當前的 \*\*「資料內容 + 排版參數快照」\*\* 並存入清單，確保後續修改拉桿不會影響已加車的標籤。

\* 支援在同一張 A4 畫布上，無縫混搭 Type A、Type B 及不同尺寸的標籤進行流式排版 (Flow Layout)。



\### 4. CSV 批次匯入 (Batch Import)

\* 內建標準 CSV 範本下載。

\* 一鍵匯入設備資料，並可指定統一尺寸與版型，瞬間轉化為待列印清單，大幅節省建檔時間。



\### 5. 高保真列印技術 (High-Fidelity Print)

\* 內建 `@media print` 列印專用樣式，列印時自動隱藏操作介面。

\* 支援「啟用高保真列印」選項，透過注入 CSS 強制指令，解決瀏覽器列印預設拔除背景色的問題，保證印出 100% 完整的工業深色風格。

\## 🛠 技術選型 (Tech Stack)

本系統採用無伺服器 (Serverless) 的純靜態網頁架構，單一 HTML 檔案即插即用，具備極高的部署便利性。



\* \*\*核心框架\*\*：HTML5 + CSS3 + \*\*Vue.js 3\*\* (CDN)

\* \*\*圖碼生成\*\*：`JsBarcode` (一維條碼) + `qrcode.js` (二維碼)

\* \*\*資料解析\*\*：`PapaParse` (CSV 解析)

\* \*\*視覺渲染\*\*：純 CSS `clip-path` 與 `linear-gradient` 繪製科技切角與警告斜線，確保列印具備完美向量清晰度。

\## 📖 部署與使用指南 (Deployment \& Usage)



\### 部署方式 (GitHub Pages 推薦)

1\. 在 GitHub 建立公開的 Repository。

2\. 將 `index.html` 檔案上傳至 `main` 分支。

3\. 進入 \*\*Settings > Pages\*\*，選擇 `Deploy from a branch` 並指定 `main` 分支的 `/ (root)`。

4\. 儲存後即可透過專屬網址永久存取。



\### 最佳列印設定 (Chrome / Edge)

為達到最完美的列印效果，請於瀏覽器按下「列印」時確認以下設定：

\* \*\*印表機\*\*：選擇「另存為 PDF」。

\* \*\*邊界 (Margins)\*\*：設定為 \*\*「無 (None)」\*\* 或 \*\*「最小」\*\*。

\* \*\*選項 (Options)\*\*：

&#x20; \* ❌ 取消勾選「頁首與頁尾 (Headers and footers)」。

&#x20; \* ✅ \*\*務必勾選「背景圖形 (Background graphics)」\*\*（確保深色背景正常輸出）。

\## ⏳ 版本沿革 (Changelog)



\* \*\*v17.0 (Current)\*\*：實作「參數快照 (Snapshot)」機制徹底隔離標籤狀態，並將尺寸選擇與加入清單按鈕整合至預覽圖下方，達到極致的所見即所得 (WYSIWYG)。

\* \*\*v13.0 - v16.0\*\*：導入「待列印清單 (購物車)」動線，支援 A/B 版型混搭列印與 CSV 批次加車功能，並優化清單顯示介面。

\* \*\*v10.0 - v12.0\*\*：新增「高保真列印」強制深色背景輸出功能；支援自訂 Logo 與警告標語；加入第一行高度佔比的精確百分比控制。

\* \*\*v5.0 - v9.0\*\*：開發「全自訂排版引擎」，解鎖外框粗細、字級大小、水平/垂直間距、以及各欄位寬度比例的即時拉桿控制，解決極端字數導致的跑版問題。

\* \*\*v2.0 - v4.0\*\*：導入 Vue 3 實現資料雙向綁定與即時預覽，整合 JsBarcode 與 QRCode.js，建立動態縮放基礎。

\* \*\*v1.0\*\*：初版靜態樣板建立，以 CSS 與 SVG 重現 Type A (橫式) 與 Type B (方型) 之 機甲視覺風格。
